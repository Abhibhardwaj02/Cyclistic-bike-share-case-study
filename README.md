# 🚲 Cyclistic Bike Share Analysis

## 📌 Project Overview

This project is based on the **Google Data Analytics Professional Certificate Capstone Case Study**. The objective is to analyze Cyclistic's historical bike-share data to understand how **casual riders** and **annual members** use the service differently and provide data-driven recommendations to increase annual memberships.

---

## 🎯 Business Task

Analyze how casual riders and annual members use Cyclistic bikes differently to help the marketing team develop strategies that encourage casual riders to become annual members.

---

## 📊 Dataset

- Source: Cyclistic Historical Trip Data (Divvy Bike Share)
- Format: CSV Files
- Data Period: Multiple quarterly trip datasets
- License: Public dataset provided for educational purposes

---

## 🔄 Data Analysis Process

### 1. Ask
- Defined the business problem
- Identified key stakeholders
- Established the business task

### 2. Prepare
- Collected and explored the datasets
- Verified data credibility using the ROCCC framework

### 3. Process
- Converted timestamps into proper datetime format
- Calculated ride duration
- Removed invalid and duplicate records
- Created additional columns (Ride Length, Day of Week)

### 4. Analyze
Performed descriptive analysis using Pivot Tables to compare:

- Ride Count by User Type
- Average Ride Length
- Average Ride Length by Day of Week
- Weekly Ride Count Trends

### 5. Share
Created visualizations and summarized key insights in a presentation.

### 6. Act
Provided business recommendations based on the analysis.

---

# 📈 Key Insights

- Members completed significantly more rides than casual riders.
- Casual riders had much longer average ride durations.
- Members mainly rode during weekdays, suggesting commuting behavior.
- Casual riders showed relatively higher activity during weekends, indicating recreational usage.

---

# 💡 Recommendations

### 1. Launch Weekend Membership Campaigns
Offer discounted annual membership promotions during weekends to target casual riders.

### 2. Promote Cost Savings
Send personalized offers to frequent casual riders highlighting the long-term savings of annual memberships.

### 3. Target Leisure Riders
Use digital marketing campaigns focused on leisure riders through social media and mobile applications.
